Performance Beef is a cloud-based farm management application that helps beef growers to automate data entry process and analyze cattle related data.

I have been a major contributor in developing this web application, ranging from initial site setup to production deployment. The application is written in JavaScirpt, PHP and MYSQL.

Some of the key features developed by me:

RESTful API

Integrated APIs with Third-party vendor to retrieve and collect day-to-day data for our iPad Scale App .

User Interface for Data Handling

Develop excel like interface using Handsontable for users to upload and edit historical data. The system will also track everyday farm's data based on user inputs. Optimize SQL queries and DB settings that have reduced response time from minutes to milliseconds.

Analytical Charts

Create interative analytics with HighCharts to provide insights on the farm's data.

Cattle Krush

Cattle Krush combines the feeder future markets and connects current corn prices and finishing live prices, giving producers, lenders, and commodity brokers the insights they need to better their opportunities of higher profits.

This real-time web application is written in Meteor.js for both frontend and backend, and MongoDB for database.

We also integrated Cattle Krush into an IOS Application using Cordova as a wrapper.
